Comment lancer un projet avec Rust?

Un peu d'explication d'abord

    Rustup installe aussi un gestionnaire de paquets nommé Cargo.
C’est quoi? Et bien cela sert à plusieurs choses mais ici on va s’en servir pour construire notre projet et le lancer.

    Cargo permet aussi de lancer des tests, gérer la documentation de votre projet et d'autres nombreuses choses!

Créons le projet !

Une seule commande !

$ cargo new hello-rust

    Rien de plus simple non?

Resultat

Nous avons maintenant un fichier avec notre projet. Il contient :

  • Cargo.toml
  • src
    • main.rs

   Cargo.toml est le fichier manifeste de Rust. C'est là que vous gardez les métadonnées pour votre projet, ainsi que les dépendances.

   main.rs est l'endroit où vous écrirez le code de notre application.

Hello world !

    A quoi ressemble le code d'un Hello world ?

fn main() {
    println!("Hello, world!");
}

    Puis on lance le projet avec:

$ cargo run
  • Maintenant ta console affiche Hello world !
  • Et voila ton premier programme en Rust !!!

Utiliser des dépendances

    Bien je veux apparaitre dans ta console. Ca tombe bien ferris-says est une dépendance qui le permet !

Comment ajouter une dépendance?

    Rien de plus simple. Dans Cargo.toml ajoutes:

ferris-says = "0.2"

    Puis utilises cette commande

$ cargo build

    Voila cargo ajoute la dépendance comme un grand!

Comment utiliser la dépendance dans le code?

    Encore une fois rien de plus simple! Tu l'importes avec :

use ferris_says::say;

Un petit exemple d'utilisation

use ferris_says::say;
use std::io::{stdout, BufWriter};

fn main() {
   let stdout = stdout();
   let message = String::from("Salut jeune Rustacé !");
   let width = message.chars().count();

    let mut writer = BufWriter::new(stdout.lock());
    say(message.as_bytes(), width, &mut writer).unwrap();
}
  • Voila maintenant je parle aussi dans ta console !
